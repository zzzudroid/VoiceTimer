# VoiceTimer

Android-приложение с голосовым управлением на русском: **таймер обратного отсчёта** и
**напоминания на естественном языке** (аналог «Помнить Всё»). Две вкладки переключаются
свайпом или нижней панелью.

## Возможности

### Таймер
- **Голосовой ввод** — «Слей воду 20 минут» или «5 минут», таймер запустится сам
- **Фоновая работа** — сигнал сработает даже при выключенном экране (ForegroundService + WakeLock)
- **Зацикленный сигнал** до ручной остановки, **Snooze** (30 сек / 1 мин / 5 мин)
- **Метка действия** над таймером и в уведомлении; индикация реального времени срабатывания

### Напоминания
- **Голосом или с клавиатуры** — «позвонить маме завтра в 9», с предпросмотром
  «Напоминание произойдёт ДД.ММ.ГГГГ (день недели) в ЧЧ:ММ»
- **Разбор дат**: «через N минут/часов/дней», «полчаса», сегодня/завтра/послезавтра, дни недели,
  части суток, точное время и AM/PM («в 8 вечера» → 20:00), даты по месяцам
  («25 декабря», «в последнюю субботу декабря»)
- **Циклические**: «каждый вторник в 12:00», «каждый день», ежедневно/еженедельно/ежемесячно/ежегодно —
  серия сама переносится на следующий период
- **Список** с вкладками Активные/Завершённые и группировкой по срокам; правка, удаление
  (с подтверждением), отметка выполнения
- Срабатывание через `AlarmManager` при закрытом приложении (точные — громкий сигнал,
  неточные — мягкий), «Отложить 10 мин», перепланирование после перезагрузки
- Расписание дня для «неточных» напоминаний и добавление в Google-календарь

### Распознавание речи
- **Облачное (Google `SpeechRecognizer`)** при наличии интернета — точнее для русского;
  при сбое — повтор и тихий откат на локальную модель
- **Офлайн (Vosk)** без сети (модель ~45 МБ вшита в APK)
- Переключается в настройках; интернет приложению не нужен — сетью занимается системный сервис

## Стек

- Kotlin + Jetpack Compose (HorizontalPager для вкладок)
- [Vosk Android](https://alphacephei.com/vosk/) — офлайн STT; системный `SpeechRecognizer` — облачный STT
- ForegroundService + WakeLock (таймер), `AlarmManager` + `BroadcastReceiver` (напоминания)
- Android API 26+

## Сборка

```bash
# Требуется: JDK 17–21, Android SDK (compileSdk 35)
./gradlew assembleDebug          # на Windows: gradlew.bat assembleDebug

# Установить на подключённый телефон
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

Модель Vosk **не хранится в git** — скачивается и распаковывается в
`app/src/main/assets/vosk-model-ru` задачей `fetchVoskModel` перед сборкой (`preBuild`).
Подтянуть вручную: `./gradlew fetchVoskModel`.

### Release

Подпись release читается из `keystore.properties` в корне проекта (в `.gitignore`):

```properties
storeFile=app/release.keystore
storePassword=...
keyAlias=voicetimer
keyPassword=...
```

```bash
./gradlew assembleRelease       # app/build/outputs/apk/release/app-release.apk
```

## Примеры голосового ввода

### Таймер
| Что сказать | Результат |
|---|---|
| «5 минут» | 5:00 |
| «1 час 20 минут» | 1:20:00 |
| «полчаса» | 30:00 |
| «Слей воду двадцать минут» | Метка «Слей воду», 20:00 |

### Напоминания
| Что сказать | Результат |
|---|---|
| «позвонить маме завтра в 9» | завтра 09:00 |
| «купить хлеб вечером» | сегодня/завтра в час «вечер» по расписанию |
| «совещание каждый вторник в 12:00» | еженедельно, ближайший вторник 12:00 |
| «поздравить деда в последнюю субботу декабря» | последняя суббота декабря |
