# Changelog

## [В разработке]

### Добавлено
- **Интервальный повтор «каждые N …»**: «полей цветок каждые три дня в 20:00»,
  «каждые две недели», «каждые 2 месяца». Поддержаны дни/недели/месяцы/годы;
  в списке и предпросмотре — подпись «каждые 3 дня» с правильным склонением.
- **Время через пробел** «в 20 00», «в 7 30» (раньше требовалось двоеточие «20:00»).

### Исправлено
- **Повтор напоминаний вообще не распознавался** («каждый день», «каждый месяц»,
  «каждую среду» и т.п. создавались как разовые): в регэкспах `\w` по умолчанию не
  матчит кириллицу. Заменено на явный класс `[а-яё]`. Заодно убран мусорный хвост
  у «ежемесячно/ежегодно» (оставалась лишняя буква в тексте напоминания).
- **Звонящее напоминание теперь можно остановить прямо в приложении**, а не только
  кнопкой в шторке уведомлений. На вкладке «Напоминания» появился баннер «Сейчас звонит»
  с кнопками «Готово» и «Отложить 10 мин»; при срабатывании приложение само переключается
  на эту вкладку.

## [1.2] — 2026-06-13

### Добавлено
- **Облачное распознавание речи** с авто-выбором движка: при интернете — системный
  Google (`SpeechRecognizer`, заметно точнее для русского), без сети — локальная модель Vosk.
  При сбое облака — повтор и тихий откат на Vosk. Переключатель в настройках
  «Облачное распознавание при интернете». Работает и в напоминаниях, и в таймере
- **Циклические напоминания**: «каждый вторник в 12:00», «каждый день в 8»,
  «каждую неделю/месяц/год», «ежедневно/еженедельно/…». После срабатывания серия
  автоматически переносится на следующий период; «Готово» не закрывает серию.
  В списке/предпросмотре — значок 🔁 с правильным склонением («каждую пятницу», «каждое воскресенье»)
- **Даты по месяцам** в разборе: «25 декабря», «пятого декабря», а также порядковый
  день недели в месяце — «в последнюю/первую субботу декабря» (с выбором ближайшего будущего года)
- **Переключение вкладок горизонтальным свайпом** (Таймер ↔ Напоминания ↔ Настройки),
  синхронно с нижней панелью; соседние страницы предзагружаются для плавности
- Подтверждение удаления напоминания (модалка) — чтобы не удалить нужное случайно

### Изменено
- Голосовой ввод напоминаний — непрерывный режим: микрофон работает как тумблер
  (можно говорить с паузами, фразы накапливаются), новый ввод затирает несохранённый черновик
- После «Сохранить» экран сразу переключается на «Все напоминания»

## [1.1] — 2026-06-13

### Добавлено
- Вкладка **«Напоминания»** — голосовые напоминания на естественном языке (аналог «Помнить Всё»):
  - Нижняя навигация: Таймер / Напоминания / Настройки
  - Экран «Новое напоминание»: распознанный текст в поле + предпросмотр
    «Напоминание произойдёт ДД.ММ.ГГГГ (день недели) в ЧЧ:ММ · через …», затем «Сохранить»
  - Разбор русских фраз: «через N минут/часов/дней», «через полчаса», «сегодня/завтра/послезавтра»,
    дни недели («в среду», «следующий понедельник»), части суток (утром/днём/вечером/ночью),
    точное время «в 18:30»/«в 9», AM/PM («в 8 вечера» → 20:00)
  - Список с вкладками Активные/Завершённые и группировкой
    Просрочено / Сегодня / Ближайшие 7 дней / Больше чем через неделю; правка, удаление, отметка
  - Срабатывание через `AlarmManager` при закрытом приложении: точные — громкий зацикленный сигнал,
    неточные — мягкий одиночный звук; кнопка «Отложить 10 мин»
  - Перепланирование напоминаний после перезагрузки устройства (`BootReceiver`)
  - Расписание дня (часы для «неточных» напоминаний) в настройках
  - Добавление напоминаний в Google-календарь (`CalendarContract`)

## [В разработке]

### Добавлено
- Иконка приложения — векторная adaptive-icon (микрофон в кольце-таймере, фон `#6750A4`),
  с поддержкой monochrome-варианта для themed icons Android 13+
- Конфигурация подписи release читается из несинхронизируемого `keystore.properties`
- Gradle-задача `fetchVoskModel` — скачивает и распаковывает модель Vosk в assets;
  вызывается автоматически перед сборкой (`preBuild`)
- Индикация реального времени срабатывания напоминания (🔔 в `ЧЧ:ММ`) под цифрами таймера —
  вычисляется как «сейчас + остаток», корректно ведёт себя при паузе/возобновлении

### Изменено
- Модель Vosk теперь вшита в APK (`assets/vosk-model-ru`) и распаковывается из assets
  при первом запуске — вместо скачивания из интернета. Приложение работает офлайн сразу
  после установки
- Модель Vosk исключена из git (`.gitignore`); тянется задачей `fetchVoskModel`

### Исправлено
- Парсинг составных числительных: «сорок девять» теперь складывается в 49.
  Раньше перед единицей захватывалось только одно слово, поэтому «два часа сорок девять минут»
  давало 2 ч 9 мин. Регулярка берёт последовательность числительных, слова отсортированы
  по длине (чтобы «девятнадцать» не матчилось как «девять»)

### Удалено
- Разрешение `android.permission.INTERNET` — приложению больше не нужен интернет

## [1.0.0] — 2026-06-09

### Добавлено
- Голосовой ввод времени и действия через Vosk (офлайн, русский язык)
  - Парсинг фраз вида «Слей воду двадцать минут», «5 минут», «1 час 30 минут», «полчаса»
  - Метка действия отображается над кольцом таймера и в уведомлении шторки
- Таймер обратного отсчёта с кольцом прогресса (цвет: зелёный → жёлтый → красный)
- ForegroundService — таймер работает при свёрнутом приложении и выключенном экране
- WakeLock — экран просыпается при срабатывании сигнала
- Зацикленный сигнал будильника (MediaPlayer, isLooping) до ручной остановки
- Кнопка «⏹ Стоп» для остановки сигнала из приложения и из уведомления шторки
- Режим «Отложить» (Snooze) с тремя вариантами: 30 сек / 1 мин / 5 мин
  - Янтарный цвет кольца в режиме snooze, таймер показывает 00:00
  - По истечении отложения сигнал повторяется снова
- Кнопки управления: Пауза / Продолжить / Сброс / Ещё раз
- Пульсирующая анимация кнопки микрофона во время записи
- Прогресс-бар загрузки русской модели Vosk при первом запуске (~45 МБ)
- Установка APK через adb напрямую с ноутбука
